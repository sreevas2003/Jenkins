# Jenkins New Item Creation (Simple)

## Overview

In Jenkins, a **New Item** is used to create a new job that performs automated tasks such as building code, running tests, or deploying applications.

---

# Steps to Create a New Item

1. Open Jenkins in a browser.

```
http://<jenkins-ip>:8080
```

2. On the Jenkins Dashboard, click:

```
New Item
```

3. Enter the job name.

Example:

```
AOSP_GERRIT_AGENT_BUILD
```

4. Select the job type:

```
Pipeline
```

5. Click:

```
OK
```

6. Configure the job if needed (pipeline script, triggers, agent label).

7. Click:

```
Save
```

---

# Result

The new Jenkins job will appear on the Jenkins dashboard and can be used to run builds automatically or manually.
